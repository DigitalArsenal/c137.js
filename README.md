# C137.js

C137.js is an es6 [Cesium](https://cesiumjs.org) distribution that is compact, offline-capable, and simple to use.

<br>

## Installation

```bash
npm install c137.js
```

<br>

## Usage

```javascript
import { Viewer } from "c137.js";
viewer = new Viewer("cesiumContainer");
```
